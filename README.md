commons-discovery Wrapper
============

OSGi Bundle wrapper for the [Apache Commons Discovery](http://commons.apache.org/proper/commons-discovery/) library.  Compatible with Apache Felix.

To add this a dependency of your project simply add the following into your Maven POM:

    <dependency>
      <groupId>com.sixdimensions.commons.osgi.wrapper</groupId>
      <artifactId>commons-discovery</artifactId>
      <version>0.5</version>
    </dependency>
